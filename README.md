# Módulo 0: Introdução ao Java

Código do "Módulo 0: Introdução ao Java" da UniPDS.

O código está organizado nas seguintes branches:

-  `main` tem o código da aula 3: variáveis, operadores e condicionais
-  `aula04-arrays-e-lacoes-de-repeticao` tem o código da aula 4: arrays e laços de repetição
-  `aula05-classes-atributos-e-objetos` tem o código da aula 5: classes, atributos e objetos
-  `aula06-metodos-e-construtores` tem o código da aula 6: métodos e construtores
-  `aula07-arrays-de-objetos-composicao-e-enums` tem o código da aula 7: arrays de objetos, composição e enums
-  `aula08-intellij-e-debug` tem o código da aula 8: IntelliJ e Debug
-  `aula09-organizando-em-arquivos-e-heranca` tem o código da aula 9: organizando em arquivos e herança
-  `aula10-pacotes-modificadores-de-acesso-getter-e-setters` tem o código da aula 10: pacotes, modificadores de acesso, getters e setters
-  `aula11a13-protected-strings-imutabilidade-StringBuilder-e-final` tem o código da aula 11, 12 e 13: protected, String, imutabilidade, StringBuilder e final
-  `aula14-praticando-com-arquivo-csv` tem o código da aula 14: praticando com arquivo CSV 
-  `aula15-praticando-com-arquivo-json` tem o código da aula 15: praticando com arquivo JSON
-  `aula16-refatoracao-e-interfaces` tem o código da aula 16: refatoração e interfaces
-  `aula17-classes-abstratas-e-static` tem o código da aula 17: classes abstratas e static
-  `aula18-object-classes-wrapper-e-javadoc` tem o código da aula 18: Object, classes Wrapper e javadoc
-  `aula19-jars-e-exceptions` tem o código da aula 19: JARs e exceptions
